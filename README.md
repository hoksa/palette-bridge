# Palette Bridge

A small, opinionated, vibe-coded tool. Helps bring your own brand/project Tailwind CSS palettes to Material Design 3 with full control over color role mappings rather than Material's black-box algorithm.

- Maps Tailwind palettes (primary, secondary, tertiary, error, neutral) to 66 M3 color roles
- Live light/dark theme preview
- Three contrast levels (Standard, Medium, High) with WCAG contrast badges
- Paste hex values, CSS variables, or OKLCH colors

**Export formats:** Kotlin (Jetpack Compose), Material JSON, Styleframe (DTCG/Figma)

**Stack:** React 19, TypeScript, Vite, Tailwind CSS 4, shadcn/ui

```bash
npm install
npm run dev
```
